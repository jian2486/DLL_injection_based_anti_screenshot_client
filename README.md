A process window anti-screenshot protection tool based on Windows Display Affinity technology and DLL injection
The DLL injected by this tool is produced by the DisplayAffinityManager project of icer233. It has been encapsulated and extended on this basis, with an improved interface and additional functions. The original project address: https://github.com/icer233/AntiScreenshotManager
